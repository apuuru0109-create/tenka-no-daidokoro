# 天下の台所（仮）

江戸時代の大坂を舞台にした、米相場生活シミュレーションRPGの試作です。

プレイヤーは小さな米問屋を継ぎ、町で情報を集め、米を仕入れ、相場を読み、
店と人々の暮らしを守りながら30日間を生き抜きます。

## Web版

[ブラウザでゲームを遊ぶ](https://apuuru0109-create.github.io/tenka-no-daidokoro/)

## 現在の段階

プリプロダクション／コアループ検証用プロトタイプです。

ローカルテストに合格した変更を`main`ブランチへ反映すると、GitHub PagesのWeb版が
自動更新される構成です。

検証する仮説は次の3点です。

1. 人物から得た情報を、相場判断へ変換する過程が面白いか
2. 利益と町の暮らしが衝突することで、判断に物語が生まれるか
3. 数字に詳しくなくても、値動きの理由を理解して上達できるか

## 起動方法

`index.html` をブラウザで開いてください。ビルドやインストールは不要です。

ローカルサーバーを使う場合:

```powershell
python -m http.server 8000
```

その後 `http://localhost:8000` を開きます。

## 簡易テスト

Node.jsが利用できる環境では、次のコマンドで構文、画面要素の参照、価格モデルを検査できます。

```powershell
node tests/smoke.mjs
```

## 操作

- 「町を歩く」で情報を1件取得します
- 現物市場で米俵を売買します
- 帳合米で三日後の値上がり・値下がりを取引します
- 「火縄大引」で終値を決めます。注文が片側へ偏った日は、反対注文か立用を選びます
- 30日終了時の資産、信用、町の暮らしで結末が決まります
